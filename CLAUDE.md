## Project Overview

This is a Python project.  It must use `uv` for package and project management with `pyproject.toml`.

## Development Guidelines
- Must use ollama 0.6.1.  Use context7 MCP when generating code related to ollama


## Development Commands

**Install dependencies:**
```bash
uv sync
```

**Add a dependency:**
```bash
uv add <package-name>
```

**Run the main script:**
```bash
uv run main.py
```
